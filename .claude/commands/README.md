# Project Slash Commands

This project provides custom slash commands for Claude Code.

- Entry point: `/modu`
- Subcommands: `/modu:project`, `/modu:plan`, `/modu:run`, `/modu:sync`, `/modu:fix`, `/modu:loop`, `/modu:feedback`, `/modu:submit`

Source of truth for requirements:

- `.modu-arena/development-plan.md`
- `.modu-arena/specs/`
- `docs/api-specification.md`
